Using extended slice syntax to reverse strings

Wednesday, October 03, 2012

^ Note to self

Python has no built in method to reverse strings (one exists for lists: list.reverse()). However, using extended slice syntax, you can reverse strings in a very novel way:

mystr = 'hello world'mystr[::-1]# > 'dlrow olleh'
